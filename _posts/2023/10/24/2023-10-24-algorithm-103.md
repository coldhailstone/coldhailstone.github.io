## 입출력 예

| s                               | return       |
|---------------------------------|--------------|
| "{{2},{2,1},{2,1,3},{2,1,3,4}}" | [2, 1, 3, 4] |
| "{{1,2,3},{2,1},{1,2,4,3},{2}}" | [2, 1, 3, 4] |
| "{{20,111},{111}}"              | [111, 20]    |
| "{{123}}"                       | [123]        |
| "{{4,2,3},{3},{2,3,4,1},{2,3}}" | [3, 2, 4, 1] |

## 풀이
```javascript
function solution(s) {
    return s
        .slice(2, s.length-2)
        .split("},{")
        .sort((a, b) => a.length - b.length)
        .reduce((acc, cur) => {
            let arr = cur.split(',');
            for (let n of arr) {
                if (!acc.includes(n)) {
                    acc.push(n);
                    return acc;
                } 
            }
        }, [])
        .map(v => parseInt(v));
}
```
