## 문제 설명
단어 `s`의 가운데 글자를 반환하는 함수, solution을 만들어 보세요. 단어의 길이가 짝수라면 가운데 두글자를 반환하면 됩니다.

## 제한사항
* `s`는 길이가 1 이상, 100이하인 스트링입니다.

## 입출력 예

| s       | return |
|---------|--------|
| "abcde" | "c"    |
| "qwer"  | "we"   |

## 풀이
```javascript
function solution(s) {
    const centerIdx = Math.floor(s.length/2);
    return s.length % 2 ? s[centerIdx] : `${s[centerIdx-1]}${s[centerIdx]}`;
}
```
