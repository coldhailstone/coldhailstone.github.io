## 문제 설명

자연수 `N`이 입력되면 1부터 `N`까지의 합을 출력하는 프로그램을 작성하세요.

## 입력설명

첫 번째 줄에 20이하의 자연수 `N`이 입력된다.

## 출력설명

첫 번째 줄에 1부터 `N`까지의 합을 출력한다.

## 입출력 예

### 입력 예 #1

-   6

### 출력 예 #1

-   21

### 입력 예 #2

-   10

### 출력 예 #2

-   55

## 풀이

```javascript
function solution(n) {
    let answer = 0;
    for (let i = 1; i <= n; i++) {
        answer += i;
    }
    return answer;
}

console.log(solution(6));
console.log(solution(10));
```
